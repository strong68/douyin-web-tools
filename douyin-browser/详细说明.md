# douyin-browser 详细说明

> 文件位置：`douyin-browser/douyin_browser.py`

---

## 一句话概述

自动打开抖音推荐页，逐条浏览视频，每次切换时从推荐接口请求中提取当前视频的 id。可通过回调函数处理每个视频 id，也支持人工交互模式。

---

## 核心流程

```
新建标签页 → 打开抖音首页
    ↓
等待并点击"推荐"入口，进入推荐页
    ↓
启动推荐接口监听（get_v2/）
    ↓
点击"下一条"按钮 → 推荐接口返回新请求 → 从 URL 解析 group_id → 即为当前视频 id
    ↓
交给回调处理（或人工按键确认）→ 继续下一条
    ↓
异常时自动重连推荐页（最多 3 次），超过则退出循环
```

---

## 类常量

| 常量 | 值 | 说明 |
|---|---|---|
| `HOME_URL` | `'https://www.douyin.com/'` | 抖音首页地址 |
| `RECOMMEND_ENTRY_XPATH` | 很长一串 XPath | 首页顶部"推荐"入口元素的定位符 |
| `NEXT_VIDEO_XPATH` | `'x://div[@data-e2e="video-switch-next-arrow"]'` | 推荐页"下一条视频"按钮的 XPath |
| `RECOMMEND_API_PATTERN` | `'get_v2/?device_platform=webapp'` | 推荐流接口的 URL 监听关键字 |

---

## 构造函数

```python
DouyinBrowser(browser: Optional[Chromium] = None)
```

- `browser`：可选，传入外部已创建的浏览器实例。不传时类内部自己创建。
- 初始化后 `self._owns_browser` 记录浏览器是否由本类创建，关闭时据此决定是否退出浏览器。

## 方法详解

### `_parser() -> argparse.ArgumentParser`

构建命令行参数解析器：

```
--max-count  最多浏览多少条视频，不传则无限循环（默认 None，不限制）
```

### `enter_recommend_page() -> None`

进入推荐页的完整流程：
1. 新建标签页
2. 每 10 次尝试刷新一次抖音首页，直到推荐入口元素出现
3. 点击推荐入口
4. 启动推荐接口监听
5. 记录到 `self.recommend_tab`

### `get_current_video_id() -> Optional[str]`

从监听到的推荐接口请求 URL 中提取视频 id：
1. 等待推荐接口响应（超时 5 秒）
2. 从 URL 中查找 `group_id=` 关键字
3. 截取到下一个 `&` 或字符串末尾

返回示例：`'7321234567890123456'`，找不到则返回 `None`。

### `next_video() -> None`

点击推荐页"下一条"按钮，等待 0.8 秒让推荐接口响应更新。找不到按钮时抛 `RuntimeError`。

### `_tab_is_alive() -> bool`

检查推荐标签页是否仍然有效（通过访问 `tab.url` 是否抛异常来判断）。

### `_is_on_recommend_page() -> bool`

判断当前标签页 URL 中是否包含 `recommend=1`（抖音推荐页的 URL 参数）。

### `_recreate_browser() -> None`

当推荐页标签失效且本类拥有浏览器时，关闭旧浏览器重建新实例。若浏览器来自外部传入，则抛 `RuntimeError`。

### `reopen_recommend_page() -> None`

从标签失效中恢复：
1. 调用 `close()` 清理资源
2. 尝试重新 `enter_recommend_page()`
3. 若失败则重建浏览器再进入

### `browse_one(callback=None) -> bool`

处理单条推荐视频：
1. 检查是否仍在推荐页，不在则返回 `False`
2. 获取当前视频 id，为空则自动切下一条（递归）
3. 若与上一条重复（连击 2 次以上），自动切下一条
4. 读到新 id 后：
   - 有回调 → 调用 `callback(video_id)`，返回 `False` 则停止
   - 无回调 → 进入人工交互（回车继续，q 退出）
5. 点击"下一条"，返回 `True`（继续循环）

### `browse_loop(callback=None, max_count=None) -> None`

持续浏览循环：
1. 进入推荐页
2. 循环调用 `browse_one`：
   - 达到 `max_count` → 打印提示并退出
   - `browse_one` 返回 `False` → 退出循环
   - 出现异常 → 重连计数器 +1，超过 3 次则退出
3. 最终调用 `close()`

### `close() -> None`

两层关闭：
1. 关闭推荐页标签
2. 若浏览器由本类创建，则退出浏览器

## 使用示例

### 命令行（人工交互模式）

```bash
# 无限浏览，直到手动按 q 退出
python douyin_browser.py

# 只看 10 条
python douyin_browser.py --max-count 10
```

### 代码调用（回调模式）

```python
from douyin_browser import DouyinBrowser
from douyin_downloader import DouyinDownloader

downloader = DouyinDownloader()

def handle_video(video_id: str) -> bool:
    print(f'获取到视频 id: {video_id}')
    result = downloader.download_video(video_id)
    print(f"下载结果: {result['message']}")
    return True  # 返回 False 可停止循环

browser = DouyinBrowser()
browser.browse_loop(callback=handle_video, max_count=20)

browser.close()
downloader.close()
```

### 组合：浏览推荐页 + 评论抓取

```python
from douyin_browser import DouyinBrowser
from douyin_comment_crawler import DouyinCommentCrawler

crawler = DouyinCommentCrawler()

def handle_video(video_id: str) -> bool:
    crawler.crawl_video_comments(video_id=video_id, max_comments=50)
    return True

browser = DouyinBrowser()
browser.browse_loop(callback=handle_video, max_count=10)
browser.close()
crawler.close()
```
