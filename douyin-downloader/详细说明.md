# douyin-downloader 详细说明

> 文件位置：`douyin-downloader/douyin_downloader.py`

---

## 一句话概述

根据视频 id 下载抖音视频或音频，支持单个下载和批量下载，自动跳过超长视频，已下载文件不重复下载，返回结果含成功率统计和退出码，可直接用于 CI/CD 或脚本自动化。

---

## 核心流程

```
输入 video_id（或多个 id 逗号分隔）
    ↓
检查本地是否已存在同名 .mp4 / .mp3 文件 → 已存在则直接返回"file already exists"
    ↓
打开视频页，监听详情接口（aweme/detail/）
    ↓
从 aweme_detail 中提取视频时长，超过 max_duration_seconds 则跳过
    ↓
获取 play_addr.url_list 候选下载地址列表
    ↓
遍历下载地址，用 requests 下载二进制文件
    ↓
根据文件魔数判断是 .mp4（视频）还是 .mp3（音频）
    ↓
保存到 output_dir/{video_id}.mp4 或 .mp3
    ↓
返回 {success, video_id, file_path, message, ...}
```

---

## 类常量

| 常量 | 值 | 说明 |
|---|---|---|
| `HOME_URL` | `'https://www.douyin.com/'` | 抖音首页地址 |
| `DETAIL_API_PATTERN` | `'aweme/detail/?device_platform'` | 视频详情接口监听关键字 |
| `DEFAULT_OUTPUT_DIR` | 脚本所在目录 | 默认下载目录 |
| `DEFAULT_RETRY_COUNT` | `3` | 单个视频下载失败时的最大重试次数 |
| `DEFAULT_MAX_DURATION_SECONDS` | `300` | 默认跳过超过 5 分钟的视频 |

---

## 构造函数

```python
DouyinDownloader(
    output_dir: Optional[str] = None,            # 下载目录，默认脚本所在目录
    browser: Optional[Chromium] = None,         # 外部传入浏览器实例，不传则自己创建
    max_duration_seconds: Optional[int] = 300   # 超过此时长则跳过下载，单位秒
)
```

初始化时：
- 创建 `requests.Session` 并配置 `User-Agent` 和 `referer` 请求头
- 自动创建输出目录

## 方法详解

### `_parser() -> argparse.ArgumentParser`

```
--video-id              单个视频 id（二选一）
--video-ids             逗号分隔的多个 id（二选一）
--output-dir            下载目录
--max-duration-seconds 超过此时长则跳过，默认 300 秒
--json                  以 JSON 格式输出完整结果
```

> `--video-id` 和 `--video-ids` 必须至少提供一个。

### `extract_video_id(video_id: str) -> str`

去除输入字符串首尾空格并返回，用于批量处理时规范化每个 id。

### `_open_browser() -> Chromium`

延迟初始化浏览器：仅在真正需要时才创建浏览器实例（`download_video` 调用时）。

### `_get_video_detail(browser, video_id) -> Dict`

获取视频详情：
1. 新建标签页，打开抖音首页
2. 启动 `aweme/detail/` 接口监听
3. 直接导航到 `https://www.douyin.com/video/{video_id}`
4. 等待接口响应，超时 15 秒
5. 关闭标签页，返回 `response.body`（已转为 dict）

### `_extract_duration_seconds(aweme_detail) -> Optional[float]`

从详情数据中提取视频时长（秒）：
- 优先取 `video.duration`
- 备选取 `duration`
- 若值大于 1000，视为毫秒，除以 1000 转换为秒
- 异常时返回 `None`

### `_download_binary(urls, base_path) -> Dict`

遍历候选下载地址直到成功：
1. 去掉 URL 中的查询参数（`?之后的`）
2. 用 `requests.Session` 发送 GET 请求（超时 30 秒）
3. 判断内容类型：
   - `content[4:8] == b'ftyp'` → `.mp4`（视频）
   - 否则 → `.mp3`（音频）
4. 写入 `base_path.mp4` 或 `base_path.mp3`
5. 返回 `{success, file_path, message}`

### `download_video(video_id: str) -> Dict`

单个视频下载的完整逻辑：

```
校验 video_id（纯数字）→ 跳过空/无效 id
检查本地是否已下载 → 已存在则返回 success=True, message="file already exists"
获取视频详情 → 失败则返回 success=False, message="failed to get video detail"
提取时长 → 超长则返回 skipped=True, message="skipped video longer than ..."
提取下载地址列表 → 为空则返回 success=False, message="download url not found"
下载二进制文件 → 更新 result 返回
```

返回字典字段：

| 字段 | 类型 | 说明 |
|---|---|---|
| `success` | `bool` | 是否成功 |
| `video_id` | `str` | 视频 id |
| `file_path` | `str` | 文件保存路径 |
| `message` | `str` | 结果描述 |
| `duration_seconds` | `float` | 视频时长（秒） |
| `skipped` | `bool` | 是否因超长被跳过 |

### `download_many(video_ids: Iterable[str]) -> Dict`

批量下载：
1. 规范化所有 id（去空格）
2. 遍历每个 id，最多重试 `DEFAULT_RETRY_COUNT`（3）次
3. 收集每个视频结果到 `results['videos']`
4. 统计 `total` / `success` / `failed`

返回结构：
```json
{
  "total": 10,
  "success": 8,
  "failed": 2,
  "videos": [
    {"success": true, "video_id": "...", "file_path": "...", "message": "..."},
    ...
  ]
}
```

### `close() -> None`

退出浏览器（若由本类创建）和关闭 `requests.Session`。

### `main() -> None`

命令行入口逻辑：
1. 解析参数，确定单个下载还是批量下载
2. 执行下载
3. 按 `--json` 参数决定输出格式
4. 以 `sys.exit(0)`（成功）或 `sys.exit(1)`（失败）退出，**支持 CI/CD 使用**

## 使用示例

### 命令行

```bash
# 下载单个视频
python douyin_downloader.py --video-id 7431772994717879567

# 批量下载多个视频
python douyin_downloader.py --video-ids 7431772994717879567,7429635203477310756

# 指定下载目录
python douyin_downloader.py --video-ids 7431...,7429... --output-dir D:\downloads

# 跳过超过 60 秒的视频
python douyin_downloader.py --video-id 7431... --max-duration-seconds 60

# 输出 JSON 结果
python douyin_downloader.py --video-id 7431... --json
```

### 代码调用

```python
from douyin_downloader import DouyinDownloader

downloader = DouyinDownloader(
    output_dir='D:/downloads',
    max_duration_seconds=300,
)

# 单个下载
result = downloader.download_video('7321234567890123456')
print(f"{result['video_id']}: {result['message']}")

# 批量下载
batch = downloader.download_many([
    '7321234567890123456',
    '7329876543210123456',
])
print(f"成功 {batch['success']}/{batch['total']}")

downloader.close()
```

### CI/CD / 脚本场景

```bash
# 在脚本中检查退出码判断是否全部成功
python douyin_downloader.py --video-ids $VIDEO_IDS
if [ $? -eq 0 ]; then
    echo "All downloads succeeded"
else
    echo "Some downloads failed"
fi
```

### 完整流水线：搜索 → 下载

```python
from douyin_searcher import DouyinSearcher
from douyin_downloader import DouyinDownloader

searcher = DouyinSearcher(max_count=5, sort='most_liked')
downloader = DouyinDownloader(max_duration_seconds=120)

result = searcher.search('搞笑视频')
video_ids = [v['aweme_id'] for v in result['videos']]
batch = downloader.download_many(video_ids)

print(f"下载完成：{batch['success']}/{batch['total']}")
searcher.close()
downloader.close()
```
