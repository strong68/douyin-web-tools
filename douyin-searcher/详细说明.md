# douyin-searcher 详细说明

> 文件位置：`douyin-searcher/douyin_searcher.py`

---

## 一句话概述

根据关键词在抖音网页版搜索视频，支持排序、发布时间、时长、范围、内容类型五种筛选条件，自动滚动翻页抓取视频 id 列表。

---

## 核心流程

```
打开抖音首页
    ↓
在搜索框输入关键词，回车
    ↓
打开筛选面板，依次点击五个筛选选项（监听接口响应）
    ↓
从接口响应中用正则提取首屏视频 id
    ↓
滚动页面，监听翻页接口，继续提取，直到达到 max_count 或触底
    ↓
返回 [{aweme_id, create_time, platform}, ...]
```

---

## 全局常量

| 常量 | 类型 | 说明 |
|---|---|---|
| `FILTER_OPTIONS` | `Dict[str, Dict[str, tuple]]` | 五个筛选维度各自的选项及其在网页上的坐标索引 `(index1, index2)`，用于点击筛选 |
| `PLATFORM_DEFAULT_FILTERS` | `Dict[str, str]` | 抖音网页未修改筛选时的默认值，用于判断是否需要调用筛选接口 |

## 构造函数

```python
DouyinSearcher(
    max_count: int = 10,                   # 最多返回多少个结果
    browser: Optional[Chromium] = None,   # 外部传入浏览器实例，不传则自己创建
    sort: str = 'latest',                 # 排序方式
    publish_time: str = 'any',            # 发布时间筛选
    duration: str = 'any',                # 视频时长筛选
    scope: str = 'unwatched',            # 搜索范围筛选
    content_type: str = 'video',         # 内容类型筛选
)
```

### 筛选参数详解

| 参数 | 可选值 | 含义 |
|---|---|---|
| `sort` | `comprehensive` | 综合排序 |
|  | `latest` | 最新 |
|  | `most_liked` | 最多点赞 |
| `publish_time` | `any` | 不限 |
|  | `day` | 一天内 |
|  | `week` | 一周内 |
|  | `half_year` | 半年内 |
| `duration` | `any` | 不限 |
|  | `under_1_minute` | 1分钟以内 |
|  | `1_to_5_minutes` | 1-5分钟 |
|  | `over_5_minutes` | 5分钟以上 |
| `scope` | `any` | 综合 |
|  | `following` | 关注的人 |
|  | `recently_watched` | 最近观看 |
|  | `unwatched` | 未看过的 |
| `content_type` | `any` | 不限 |
|  | `video` | 视频 |
|  | `image_text` | 图文 |

## 方法详解

### `_parser() -> argparse.ArgumentParser`

构建命令行参数解析器，参数与构造函数对应。

```
--keyword        搜索关键词（必填）
--max-count      最多返回数量，默认 10
--json           输出完整 JSON 结果
--sort           排序方式
--publish-time   发布时间
--duration       视频时长
--scope          搜索范围
--content-type   内容类型
```

### `_parse_first_page(raw_text: str) -> List[Dict]`

从筛选接口响应的原始 HTML 文本中，用正则 `"aweme_info":\s*{.*?"cover":` 切出每个视频块，再提取 `aweme_id` 和 `create_time`。用于首屏数据解析。

### `_parse_more_pages(data: List[Dict]) -> List[Dict]`

从翻页接口返回的 JSON `data` 列表中，读取每个元素的 `aweme_info` 提取视频信息。用于滚动翻页数据解析。

### `_normalize_filter_value(group, value) -> str`

将传入的筛选值标准化（小写、去空格、`-` 替换为 `_`），并校验是否在 `FILTER_OPTIONS` 可选范围内，不合法则抛 `ValueError`。

### `_normalize_filter_config(**kwargs) -> Dict[str, str]`

批量标准化全部五个筛选参数。

### `_filter_option_xpath(index1, index2) -> str`

根据筛选坐标 `(index1, index2)` 生成对应选项的 XPath：
```
x://span[@data-index1="0" and @data-index2="1"]
```

### `_open_filter_panel(tab) -> None`

反复移动鼠标触发悬浮，直到筛选面板内的选项元素出现（最多 50 次尝试），超时抛 `RuntimeError`。

### `_apply_single_filter(tab, group, value) -> Optional[Any]`

点击单个筛选选项：启动 `general/search/stream/` 接口监听 → 点击选项 → 等待响应 → 返回响应对象。若选项已有选中样式（`u39cEW99`）则跳过点击返回 None。

### `_apply_filters(tab) -> Optional[Any]`

依次调用 `_apply_single_filter` 应用五个维度的筛选，返回最后一个有意义的响应。

### `_open_browser() -> Chromium`

返回 `self.browser`，由构造函数保证已初始化。

### `close() -> None`

若浏览器由本类创建，则退出浏览器；无论是否创建都置 `self.browser = None`。

### `search(keyword: str) -> Dict[str, Any]`

完整搜索流程：

1. 参数校验（keyword 非空）
2. 打开浏览器 → 新建标签页
3. 进入抖音首页，在搜索框输入关键词并回车
4. 调用 `_apply_filters` 应用筛选，从接口响应解析首屏视频
5. 若结果数不足 `max_count`，启动翻页监听，滚动页面持续抓取
6. 最多保留 `max_count` 条，去重后返回

返回结构：
```json
{
  "keyword": "搞笑视频",
  "total": 10,
  "videos": [
    {"aweme_id": "7321...", "create_time": "1743...", "platform": "douyin"},
    ...
  ],
  "filters": {"sort": "latest", "publish_time": "any", ...}
}
```

## 使用示例

### 命令行

```bash
# 基本搜索，打印 video_id 列表
python douyin_searcher.py --keyword "性感舞曲"

# 搜索 20 条，输出 JSON
python douyin_searcher.py --keyword "性感舞曲" --max-count 20 --json

# 只看一周内、1-5 分钟的视频
python douyin_searcher.py --keyword "舞蹈" --publish-time week --duration 1_to_5_minutes
```

### 代码调用

```python
from douyin_searcher import DouyinSearcher

searcher = DouyinSearcher(
    max_count=20,
    sort='latest',
    publish_time='week',
    content_type='video',
)

result = searcher.search('搞笑视频', sort='latest', publish_time='week', content_type='video')
for video in result['videos']:
    print(video['aweme_id'])

searcher.close()
```

### 组合使用（搜索 → 下载）

```python
from douyin_searcher import DouyinSearcher
from douyin_downloader import DouyinDownloader

searcher = DouyinSearcher(max_count=5, sort='most_liked')
downloader = DouyinDownloader()

result = searcher.search('美食')
video_ids = [v['aweme_id'] for v in result['videos']]
downloader.download_many(video_ids)

searcher.close()
downloader.close()
```
